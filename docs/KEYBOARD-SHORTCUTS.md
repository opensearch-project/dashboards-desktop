# Keyboard Shortcuts

Press **?** in the app to show this panel.

---

## Chat

| Shortcut | Action |
|----------|--------|
| Cmd+K / Ctrl+K | Open or focus chat panel |
| Cmd+Shift+Enter | Toggle fullscreen chat |
| Cmd+N / Ctrl+N | New conversation |
| Enter | Send message |
| Shift+Enter | New line in message |
| Escape | Close chat panel |
| Up Arrow | Edit last message (when input is empty) |

## Models

| Shortcut | Action |
|----------|--------|
| Cmd+M / Ctrl+M | Command palette / model switcher |
| `/model <name>` | Switch model in chat |

## Navigation

| Shortcut | Action |
|----------|--------|
| Cmd+, / Ctrl+, | Open settings |
| Cmd+1-9 / Ctrl+1-9 | Switch workspace by position |
| Cmd+[ / Ctrl+[ | Toggle sidebar |

## General

| Shortcut | Action |
|----------|--------|
| Cmd+Shift+I | Open DevTools |
| Cmd+R / Ctrl+R | Reload OSD UI |
| Cmd+Q / Ctrl+Q | Quit app |
| ? | Show shortcuts help panel |
